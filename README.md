# 💳 Loan Default Prediction App

A Streamlit-based machine learning app that predicts the probability of loan default using borrower financial and demographic details.

---

## 🚀 Features
- Real-time default prediction  
- Probability-based risk assessment  
- Simple and interactive UI  
- End-to-end ML pipeline  

---

## 🛠️ Tech Stack
Python • Pandas • Scikit-learn • Streamlit • Joblib  

---

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
